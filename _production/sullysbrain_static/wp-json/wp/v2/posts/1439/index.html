Over the next serveral weeks I will be diving into my work in creating a personal assistant, much like a private chatGPT. The main reason will be so I can explore the different capabilities of LangChain and how I can use it to do exploratory data analysis on my library of PDF files, emails, and other offline documents.<\/p>\n\n\n\n

As a quick proof of concept, I used Streamlit to create a very crude web interface that interacts with the OpenAI and Wikipedia APIs to have the Large Language Model (LLM) respond to my prompts.<\/p>\n\n\n\n

As you can see in the video below, even though it is using the OpenAI model (which was trained during the past few years and has a knowledge cutoff of 2021), the model is able to accurately respond to recent news events of India’s successful (and impressive) moon landing. Had I only used ChatGPT, it would not know about these recent event and would reply about having the 2021 cutoff of it’s knowledge.<\/p>\n\n\n\n

I’m looking forward to sharing the deep dives as I create this application and bring it online. All the code for this will be available at my github page at https:\/\/github.com\/sullysbrain<\/a> for you to view and watch the progress.<\/p>\n\n\n\n

Coding the Bare Bones<\/h2>\n\n\n\n

To begin, I imported both Streamlit and Langchain using python. Streamlit will allow me to create the web interface in just a few lines of code. For production, we’ll have to build out the foundation a bit, but for proof of concept, this will allow me to have a very rough working prototype.<\/p>\n\n\n\n

import streamlit as st\n\nfrom langchain.utilities import WikipediaAPIWrapper\nfrom langchain.agents import load_tools, initialize_agent, AgentType\nfrom langchain.llms import OpenAI \n\nimport os\nimport sys\nsys.path.append('_private\/')\nfrom _private import api\nos.environ['OPENAI_API_KEY'] = api.API_KEY<\/code><\/pre>\n\n\n\n
I also imported my API key in order to pipe LangChain into OpenAI’s API. This is saved as a separate file for security. The next step will be to setup our LLM with OpenAI and initialize the LangChain tools. I’m selecting Verbose=True in order to see exactly what it is thinking in the terminal as it attempts to answer my questions. This will help me troubleshoot later, in case I need to add some subtle prompting in code to guide OpenAI’s results.<\/p>\n\n\n\n
# Init the LLM and Tools\nllm = OpenAI(temperature=0)\ntools = load_tools(['wikipedia'], llm=llm)\nagent = initialize_agent(tools, llm, \n                         agent=AgentType.ZERO_SHOT_REACT_DESCRIPTION, verbose=True)<\/code><\/pre>\n\n\n\n
The next step will be to use Streamlit to create the web interface. It is running locally on my own computer for now so I can see it and interact with it via a web browser. <\/p>\n\n\n\n
# collect user prompt\nst.title('SullyGPT')\ninput = st.text_input('Enter a prompt:')\n\nif input:\n    text = agent.run(input)\n    st.write(text)<\/code><\/pre>\n\n\n\n
